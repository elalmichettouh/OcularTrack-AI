# OcularTrack-AI

A high-performance, real-time ocular telemetry and anomaly detection engine powered by Computer Vision. This system builds a localized coordinate system for tracking pupil dynamics, measuring angular velocity, and capturing micro-saccadic anomalies.

## Key Features
- Localized Coordinate Mapping
- Angular Velocity Tracking
- Real-Time Anomaly Detection
- Automated Telemetry Logging

## Tech Stack
- Python
- OpenCV
- NumPy
- CSV Logging
